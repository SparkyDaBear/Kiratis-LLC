# Asset Structure

Use this folder for all design and media exports used by the site.

For detailed shape, geometry, theme, and palette rules, see [docs/vector-asset-spec.md](../docs/vector-asset-spec.md).

## Recommended exports

- `.ai` for Illustrator source files.
- `.svg` for logos, icons, glyphs, ornaments, and simple illustrations.
- `.webp` for painterly artwork, textured hero imagery, and background art.
- `.png` only when transparency is needed and SVG is not suitable.
- `.jpg` for photo-based images, if used.
- `.pdf` for archive or handoff exports.

## Folder map

- `logos/` for wordmarks, emblems, and site brand marks.
- `icons/` for small interface symbols and utility marks.
- `illustrations/hero/` for large decorative hero art.
- `illustrations/sections/` for supporting illustrations used in content blocks.
- `illustrations/ornaments/` for small flourishes, borders, glyphs, and dividers.
- `textures/` for grain, paper, wash, and other surface treatments.
- `backgrounds/` for full-bleed scenic or abstract background assets.
- `photos/consulting/` for consulting-related images.
- `photos/junk-removal/` for junk removal photos.
- `photos/computer-builds/` for build photos and product shots.

## Naming

Use short, descriptive lowercase names with hyphens, for example:

- `logo-primary.svg`
- `hero-alchemical-wash.webp`
- `divider-moonline.svg`
- `texture-parchment.webp`
